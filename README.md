# 🌼 A Vida Como Ela É — Conscientização sobre Endometriose

## 📖 Sobre o Projeto

Projeto institucional desenvolvido para a iniciativa A Vida Como Ela É, voltada à conscientização sobre endometriose e ao acolhimento de mulheres que convivem com a condição.

O objetivo foi criar uma presença digital acessível, organizada e acolhedora, capaz de apresentar a causa, divulgar ações realizadas, disponibilizar conteúdos educativos e facilitar o contato com a comunidade.

---

## 🎯 Problema

Muitas mulheres convivem durante anos com sintomas relacionados à endometriose sem acesso adequado à informação, acolhimento ou orientação.

O desafio foi criar uma experiência digital capaz de:

- Organizar o conteúdo de forma simples;
- Facilitar a navegação;
- Valorizar o acolhimento;
- Destacar as ações realizadas pelo projeto;
- Criar canais de contato acessíveis.

---

## 💡 Solução Proposta

Foi desenvolvida uma landing page institucional responsiva utilizando:

- HTML5
- CSS3
- JavaScript

A solução inclui:

- Apresentação institucional do projeto;
- Galeria de palestras;
- Participações em mídia;
- Redes sociais;
- Formulário de contato;
- Integração com WhatsApp.
